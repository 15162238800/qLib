Welcome Houdinistas!

You can download qLib from here. But it's better to use Git!

Installing Git:

  • On Linux use your distributions package manager.
  • On OS X use Git for OS X.
  • On Windows use Git for Windows, which comes with an acceptable shell also (bash).

Open a shell and clone the repository:

$ git clone https://github.com/qLab/qLib.git

Later if you want to update the library just go into your cloned repository and run the following:

$ cd qLib
$ git pull

Thats it!
