Welcome Houdinistas!

You can download qLib from here.

But it's better to use git! Grab git from here for OS X, here for Windows (that one has a decent bash shell, too) or from your package manager on Linux.

Open a shell and clone the repository:

$ cd <target folder>
$ git clone https://github.com/qLab/qLib.git

Later if you want to update the library just go into your cloned repository and run the following:

$ cd qLib
$ git pull

Thats it! You just updated to the latest version.

qLib is maintained by the qLab organization. ("Tactile" theme by Jason Long, from GitHub Pages)
